Thanks for your interest in contributing!

This project is a React-based frontend dashboard. Contributions that improve UI, functionality, performance, or documentation are welcome.

---

## Tech Stack

* React
* JavaScript
* CSS
* React Hooks

Please keep contributions consistent with the existing stack and structure.

---

## Getting Started

1. **Fork** the repository
2. **Clone** your fork:

   ```bash
   git clone https://github.com/your-username/student-management-system.git
   ```
3. Install dependencies:

   ```bash
   npm install
   ```
4. Start the app:

   ```bash
   npm start
   ```

---

## Contribution Guidelines

* Create a new branch for each change:

  ```bash
  git checkout -b feature/your-feature-name
  ```
* Keep components reusable and clean
* Follow the existing dark UI theme
* Avoid unnecessary dependencies

---

## Known Limitations

The following features are **UI-only and not functional yet**:

* Settings
* Notifications
* Admin Profile
* Logout

Contributions to implement these are welcome.

---

## Pull Requests

* Push changes to your fork
* Open a Pull Request against `main`
* Clearly describe what was changed and why
* Ensure the app runs without errors

---

## Issues & Suggestions

* Use GitHub Issues for bugs or feature requests
* Include screenshots or clear steps when possible
